# Palabras Encadenadas

Aplicación web hecha con React y Vite para jugar a "Palabras Encadenadas". El juego propone formar una cadena de palabras válidas: cada nueva palabra debe empezar con la última letra de la anterior, no puede repetirse y suma puntos según su longitud.

## Requisitos

- Node.js instalado.
- Un gestor de paquetes compatible con `npm`.

## Cómo correr el proyecto

1. Instalar dependencias:

```bash
npm install
```

2. Levantar el servidor de desarrollo:

```bash
npm run dev
```

3. Abrir la URL que muestra Vite en el navegador. Por defecto es `http://localhost:5173`.
	Si tu entorno usa otro puerto, revisá la configuración en `.env`.
